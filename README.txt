WALKTHROUGH:
1. Buat file index dilajut installasi TailwindCSS
2. Buat 'Main Gallery' terlebih dahulu.
3. Buat 'Footer'.
4. Buat 'Header', hias dengan animasi dan teks yg bisa menarik minat pengunjung halaman.
5. Buat fitur untuk mengatur berapa gambar per kolom.
6. Buat fitur 'Load More'.
7. Buat fitur 'Expand Gambar' ketika 'card' diklik.

FITUR YG SEDANG DIKEMBANGKAN:
-> Header Landing dengan Background Gambar nge-Slide sendiri.
-> Tombol Load More supaya Gambar Tidak ter-Load semua ketika user ingin mengunjungi bagian Footer.
-> Tombol untuk mengontrol gambar, mau 4 gambar per column, 2 gambar per column, terserah user. Tp ada default-nya, dari monitor lebar 4 gambar per column, sampai android 1 gambar per column.
-> Animasi buat tulisan yg di header..
